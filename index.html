ZXing Decoder Online

Decode a 1D or 2D barcode from an image on the web. Supported formats include:

  • UPC-A and UPC-E
  • EAN-8 and EAN-13
  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar
  • RSS-14 (all variants)
  • RSS Expanded (most variants)
  • QR Code
  • Data Matrix
  • Aztec
  • PDF 417
  • MaxiCode
